### Test

https://golang.org/pkg/testing/

Golang은 별도의 라이브러리나 프레임워크 도움없이 go test 만으로도 테스트한다.

테스트 코드는 테스트하고자하는 파일명 + _test.go 로 작성하면 된다.
테스트 파일은 테스트 대상파일과 같은 패키지 에 넣으면 된다.
